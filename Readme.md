# Git Basics

## Git CLI & Remote

- `git init`
- `git status`
- `git add <filename>`
- `git commit -m "Commit message"`
- `git push -u origin <branch-name>`
- `git remote -v`
- `git remote add origin <ssh link>`
- `git clone <ssh link>`

## Git Branches & PRs

- `git switch -c <branch-name>`
- `git switch <branch-name>`
- `git pull`
- `git branch -d <branch-name>`
- `git branch`
